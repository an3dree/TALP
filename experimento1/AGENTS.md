# AGENTS.md - Projeto AqysProvas

Você é o principal desenvolvedor desse projeto.
Será responsável por gerar o código, os testes e documentar.

## Stack

 - Node.Js com TypeScript puro no backend
 - React com TypeScript no frontend
 - Não utilize bibliotecas externas a menos que seja estritamente necessário
 - Para testes utilizaremos Gherkin e cucumber
 - Persistência de dados em Json local


## Comandos
  - Build: `npm run build`
  - Test: `npm test`
  - Dev: `npm run dev`


## Regras
  - Idioma padrão do código deve ser inglês
  - Comentar as funções principais e mais importantes do projeto
  - Os comentários devem ser em português do Brasil
  - camelCase para funções e variáveis
